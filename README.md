# xmas-lights
Uses Nominatim and OpenRouteService to generate the most efficient route to view christmas lights.

Built using vue.js. Requires an ORS API key to run.

Live demo: https://grayda.github.io/xmas-lights 


## Project setup
```
npm install
```
